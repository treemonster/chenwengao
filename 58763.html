html标签中唯一可以实现内容垂直居中的元素只有td，实现原理是使用vertical-align: middle;这个样式，或者valign="middle"属性。这是因为valign属性只有td上有效果，而vertical-align属性必须作用于非块级元素（即使display:inline-block;都不行）。

不过自从ie8支持display:table-cell样式规则后，所有标签都可以通过样式来做到这一点了。 

<head>
<meta charset='utf-8'/>
<meta http-equiv="X-UA-Compatible" content="IE=8">  
</head>
<style>
body,html{
  width:100%;
  height:100%;
  margin:0;
}
div{
  position: relative;
}
#a{
  background: #9f9;
  vertical-align: middle;
  display: table-cell;
  text-align: center;
}
#b{
  background: #f99;
  margin: 0 auto;
  display: block;
}
</style>
  <div id='a'>
    <div id='b'></div>
  </div>
<script>
var o=0,p=Math.PI;
setInterval(function(){
  if(o++===360)o=0;
  with(a.style)
    width=Math.sin(o/180*p)*100+300+'px',
    height=Math.cos(o/180*p)*100+300+'px';
  with(b.style)
    width=Math.cos(o/180*p)*50+100+'px',
    height=Math.sin(o/180*p)*50+100+'px';
},1);
</script>

有一个缺点是这种方式的元素不能通过百分比的值来设置有效的宽高，解决办法是在他外面套一个display:table的标签，在这个标签上使用百分比的方式来设置宽高。虽然这样做会增加一个额外标签，但是这是目前唯一的解决办法了。 

<head>
<meta charset='utf-8'/>
<meta http-equiv="X-UA-Compatible" content="IE=8">  
</head>
<style>
body,html{
  width:100%;
  height:100%;
  margin:0;
}
div{
  position: relative;
}
#a{
  background: #9f9;
  vertical-align: middle;
  display: table-cell;
  text-align: center;
}
#b{
  background: #f99;
  margin: 0 auto;
  display: block;
}
#c{
  display:table;
  width:100%;
  height:100%;
}
</style>
<div id='c'>
  <div id='a'>
    <div id='b'></div>
  </div>
</div>
<script>
var o=0,p=Math.PI;
setInterval(function(){
  if(o++===360)o=0;
  with(a.style)
    width=Math.sin(o/180*p)*100+300+'px',
    height=Math.cos(o/180*p)*100+300+'px';
  with(b.style)
    width=Math.cos(o/180*p)*50+100+'px',
    height=Math.sin(o/180*p)*50+100+'px';
},1);
</script>
