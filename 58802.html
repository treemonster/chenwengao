centos7提供了minimal的版本，镜像只有700MB不到，非常小巧，因此我开始使用这个版本来进行本地开发。

centos7 官网 https://www.centos.org/

centos7 minimal 镜像下载地址

http://mirrors.zju.edu.cn/centos/7/isos/x86_64/CentOS-7-x86_64-Minimal-1511.iso

我使用virtualbox作为本地虚拟机，安装镜像过程需要注意以下几点：

  1. 需要启用eth0，否则安装完成之后无法上网
  2. 语言最好使用english，中文也可以，不过后面可能会遇到乱码问题

安装成功后，依次执行以下命令进行桌面安装：


yum groupinstall "X Window System"

yum install gnome-classic-session gnome-terminal nautilus-open-terminal control-center liberation-mono-fonts

unlink /etc/systemd/system/default.target

ln -sf /lib/systemd/system/graphical.target  /etc/systemd/system/default.target

reboot

桌面安装完成之后，可以方便的在图形界面下安装增强功能了。

首先点vmbox工具条的“设备”，“安装增强功能”。图形界面下会自动挂载一个vmbox的光盘，然后按照提示运行即可安装。

遇到提示 bzip2 命令不存在的话，使用以下命令先安装bzip2，然后再次运行光盘。

yum install bzip2

安装完毕之后重启即可。增强功能包括共享文件夹，自动调整分辨率等。
