https://github.com/hwdsl2/setup-ipsec-vpn/issues/605

Edit /etc/ipsec.conf and replace sha2-truncbug=yes with sha2-truncbug=no, save the file and run sudo service ipsec restart. Now try to re-connect.

另外android上这个标记为no可能会连不上，参考：

https://github.com/hwdsl2/setup-ipsec-vpn/issues/391

反正目前以iphone使用为主，不考虑安卓的情况，ios13之前的版本是兼容yes的，不知道为何这个版本去掉了。
