拉下来这个项目

git clone https://github.com/oblique/create_ap.git

默认的create_ap代码使用不安全的wpa加密，需要设置为wpa2加密。修改 create_ap 文件（auth_algs和wpa=2是新增的）

    cat << EOF >> $CONFDIR/hostapd.conf
auth_algs=3 # instead of 1
wpa=2 # ${WPA_VERSION}
wpa_${WPA_KEY_TYPE}=${PASSPHRASE}
wpa_key_mgmt=WPA-PSK
wpa_pairwise=TKIP CCMP
rsn_pairwise=CCMP
EOF
fi

启动热点

./create_ap wlan0 eth0 Treemonster-HomeServer JHBUIbHtFtr788gy
