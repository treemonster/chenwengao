今天写了这样的sql语句： select id,title, (select name from tab_name where tab_name) as _name from contents where _name='test'; 然后mysqld喜闻乐见的报错了：<font color=red>[Err] 1054 - Unknown column '_name' in 'where clause'</font>

搜了一下网上的说法，得出的结论是where不能取别名。因此我修改了另一种写法：

select id,title, (select name from tab_name where tab_name) as _name from contents group by id having _name='test';
