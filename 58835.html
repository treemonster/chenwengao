由于mac上找不到好用的免费压缩包解压工具，要么收费要么垃圾，索性在虚拟机里装了。

以下安装命令：

cd /tmp
wget https://jaist.dl.sourceforge.net/project/p7zip/p7zip/16.02/p7zip_16.02_src_all.tar.bz2
mv download p7zip.bz2
tar -xvf p7zip.bz2
cd p7zip
make install

然后就可以使用 7za x 命令解压7z文件了。

例如：

7za x PCSE00442【乐高蝙蝠侠】美版.7z
