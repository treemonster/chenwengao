应脑残设计师的要求，需要给轮播图加上触摸交互。于是乎网上搜了bootstrap自带轮播图支持触摸的操作，没想到还真找到了，而且还有多个方案。下文稍作整理。

参考网址： http://www.tuicool.com/articles/7zu6Vj6

方案1 jquery mobile

http://jquerymobile.com/download/ 引用jquery.mobile.js后添加以下代码:

 $("#carousel-generic").swipeleft(function() {
   $(this).carousel('next');
 }).swiperight(function() {
   $(this).carousel('prev');
 });

方案2 TouchSwipe jquery plugin

https://github.com/mattbryson/TouchSwipe-Jquery-Plugin

引用 jquery.touchSwipe.js ，并添加以下代码片段：

$("#carousel-generic").swipe({
   swipeLeft: function() { $(this).carousel('next'); },
   swipeRight: function() { $(this).carousel('prev'); },
});

方案3 hammer.js & jquery.hammer.js

hammer.js (http://eightmedia.github.io/hammer.js/) +

jquery.hammer.js (https://github.com/EightMedia/jquery.hammer.js)

非amd情况下需要依次引用hammer.js和jquery.hummer.js。

然后添加如下代码片段：

$('#carousel-generic').hammer().on('swipeleft', function(){
  $(this).carousel('next');
}).on('swiperight', function(){
  $(this).carousel('prev');
});

个人推荐方法2，因为引用代码较少。其次方法3。不推荐方法1，因为jquery mobile部分情况下会与bootstrap冲突。
