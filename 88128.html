本文简要记录端口相关操作。

  1. 查看iptables状态

    iptables -L

  2. 开启一个端口立即（临时）生效，-I表示写入到现有规则之前，如果-I换成-A表示写在现有规则之后。顺序很重要，因为有些规则是禁止端口访问的，如果在禁止规则之后添加那么可能就无效了。

    iptables -I INPUT -p tcp -m state --state NEW -m tcp --dport 22 -j ACCEPT

  3. 关闭一个端口立即（临时）生效

    iptables -D INPUT -p tcp -m state --state NEW -m tcp --dport 22 -j ACCEPT

  4. 永久生效 把以上命令写入 /etc/sysconfig/iptables文件，并重启iptables服务

    service iptables restart
